# 🐍🤖 Curso Python + Inteligência Artificial Do Zero!

Bem-vindo ao repositório oficial do curso gratuito de **Python + Inteligência Artificial** ministrado no canal [AloiTech](https://www.youtube.com/@aloitech76).

O objetivo deste curso é sair da teoria de "conversar com a IA" e usar a programação para automatizar problemas reais da criação de conteúdo! Com ferramentas open-source, vamos rodar inteligências fantásticas de forma local no seu PC.

## 🚀 O que vamos construir?
Durante nosso projeto prático, você aprenderá a criar do zero:
- 🎙️ Extrator de áudio e gerador de legendas automáticas.
- 💬 Analisador de sentimento dos comentários do YouTube.
- 🌍 Tradutor inteligente de textos e vídeos.
- 🛠️ Gerador de opções de títulos para testes A/B.

## 🧰 Nossas Ferramentas (Stack)
- **Linguagem Principal:** Python 3.x
- **Manipulação de Mídia:** FFmpeg, Pydub
- **Transcrição de Áudio:** OpenAI Whisper
- **LLMs e Processamento Local:** Meta Llama, Ollama
- **Integrações Web:** YT-DLP

## 📂 Estrutura do Curso
*(Os códigos-fonte serão adicionados neste repositório conforme as aulas forem ao ar)*
