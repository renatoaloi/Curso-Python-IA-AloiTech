# Aula 03: Transcrição de Áudio com Inteligência Artificial (Whisper)

Nesta aula, daremos nossos primeiros passos no mundo da automação com IA local, construindo um script que extrai o áudio de um vídeo e transcreve tudo em texto usando o **OpenAI Whisper**!

## 🎯 Objetivos da Aula
1. Entender e criar um **Ambiente Virtual (Venv)**.
2. Instalar e configurar o **FFmpeg** (para lidar com arquivos de mídia).
3. Instalar o **Whisper** (IA da OpenAI para transcrição).
4. Escrever um script em Python que faz o processo completo de ponta a ponta.

## 🧰 Ferramentas Utilizadas
- **Python 3.x**
- **FFmpeg**: Biblioteca externa para processamento de áudio e vídeo.
- **Whisper (OpenAI)**: Modelo de IA para reconhecimento de fala.

## 📂 Arquivos do Projeto
- `main.py`: Script principal que orquestra a extração e transcrição.
- `video_teste.mp4`: Vídeo de exemplo utilizado para demonstração.

## 🚀 Como Executar
1. Certifique-se de ter o Python e o FFmpeg instalados em sua máquina.
2. Crie e ative um ambiente virtual.
3. Instale as dependências necessárias (`pip install -r requirements.txt`).
4. Coloque um arquivo de vídeo chamado `video_teste.mp4` na mesma pasta do script.
5. Execute o script: `python main.py`
