# Aula 02: Primeiros Passos com Python

Nesta aula, vamos preparar o terreno para a nossa jornada na automação com Inteligência Artificial, realizando a instalação do Python do absoluto zero e escrevendo nossos primeiros comandos e scripts!

## 🎯 Objetivos da Aula
1. Baixar e instalar o **Python** no computador corretamente.
2. Entender como abrir e usar o **Console Interativo do Python**.
3. Escrever e rodar os **primeiros comandos** básicos.
4. Criar, salvar e executar o seu **primeiro script** (`.py`).

## 🧰 Ferramentas Utilizadas
- **Python 3.x**: A linguagem de programação principal que rodará nossas IAs futuramente.
- **Terminal (Prompt de Comando ou PowerShell)**: A "tela preta" onde nossos programas ganharão vida.
- **Editor de Código / Bloco de Notas**: Onde escreveremos os scripts (neste curso, usaremos o VS Code).

## 📂 Arquivos do Projeto
- `primeiro_script.py`: Arquivo de roteiro inicial contendo nosso primeiro código de exemplo em Python.

## 🚀 Como Executar
1. Durante a instalação do Python, lembre-se de marcar a caixinha **"Add python.exe to PATH"**.
2. Abra o seu Terminal ou Prompt de Comando.
3. Para abrir o console interativo, digite apenas `python` e aperte Enter (para sair, digite `exit()`).
4. Para executar o seu arquivo de código salvo, navegue até a pasta pelo terminal e digite: 
   ```bash
   python primeiro_script.py
   ```
