# Aula 01: Introdução à Automação com IA Local

Bem-vindos ao início da nossa jornada! Nesta primeira aula do curso de **Python + Inteligência Artificial Do Zero**, você vai descobrir a proposta do nosso projeto e qual será a nossa abordagem: fugir do uso "comum" de IA em forma de chat e partir para a prática, focando em automação de tarefas reais.

Esta aula é essencialmente teórica e desenhada para alinhar expectativas e apresentar nossa stack tecnológica. **Não teremos código-fonte nesta aula!**

## 🎯 Objetivos da Aula
1. Entender como usar o **Python** como a melhor ponte para se conectar com a Inteligência Artificial autônoma.
2. Compreender a diferença entre utilizar a **IA via "chat"** vs **integrar IA locais/LLMs autônomos** (para que ela trabalhe sozinha para você).
3. Conhecer os detalhes das **ferramentas Open-Source** (gratuitas) que vão impulsionar nosso projeto.
4. Visualizar todos os **exemplos práticos** baseados na criação de conteúdo no YouTube que faremos nos próximos vídeos.

## 🧰 Nossas Ferramentas (Stack que será Ensinada)
As tecnologias listadas abaixo serão a base dos nossos robôs autônomos durante este curso:
- **Python 3.x**: O cérebro da extração e lógica da automação.
- **FFmpeg e Pydub**: O "motor" super-leve que permitirá a manipulação de vídeo e extração de áudio.
- **OpenAI Whisper**: Nossa ferramenta mágica para transcrever minutos inteiros de áudio.
- **Llama / Ollama / LLM Locais**: Os modelos de raciocínio de alto potencial que rodaremos *offline*, no seu próprio PC.

## 📂 Arquivos do Projeto
Como esta é apenas a **Apresentação do Curso**, não teremos códigos (`.py`) ou pacotes para download por enquanto. Use este bloco de tempo para relaxar, absorver a lógica de cada engrenagem tecnológica e se preparar.
